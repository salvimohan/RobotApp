###### setup

1. Clone this repo:
    git clone https://github.com/salvimohan/RobotApp.git

2. Install the dependency (make sure ruby installed in your machine. ruby version 2.6.5)
    
3. Change to the app directory:
    cd RobotApp

4. Install dependencies:
    bundle install

5. Running the app:
	ruby index.rb

6. Running the tests:
	bundle exec rspec
